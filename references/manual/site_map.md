# 2050@2026 人工地图层

本文件来自用户提供的会场地图截图，由人工阅读整理。不要把它当作精确 GIS 坐标；它用于路线推荐时判断大致方位、跨区成本、餐饮补给和休息点。

## 地图主结构

- 主会场位于云栖小镇国际会展中心周边，核心活动块集中在地图下方中部。
- 图中核心色块从西向东大致为：D 区、C 区、B 区、A 区。
- A 区在东南侧，靠近水系和星月街一侧；B 区在 A 区西侧；C 区在 B 区西侧；D 区在更西北侧。
- A/B/C/D 之间可以视作同一主会场簇，但连续换场仍应预留步行和找入口时间。
- 地图东侧有大片公园/绿地，适合低强度休息、散步、露营或户外活动相关判断。
- 地图上北侧和东西侧分布大量餐饮点，午晚餐推荐不要只看活动内容，也要留出离场与排队时间。

## 街道与方位

- 北侧主路：河山街。
- 南侧主路：星月街。
- 东侧可见科海路一带。
- 中央主会场附近有多条支路，首次到场者需要按区域和楼层找入口，不建议安排零间隔连续活动。

## 场地块人工理解

- A 区：主会场东南侧，适合安排会议厅、论坛、集中活动后的下一站；靠近水系一侧，跨到户外区域成本较低。
- B 区：主会场中部衔接区，适合做路线中转点。
- C 区：主会场西侧较大的黄色块，适合作为从 D 区/餐饮区进入主会场后的第一站。
- D 区：主会场西侧蓝色块，靠近餐饮和酒店/商业配套一侧，适合作为休息、补给和切换点。
- 东侧绿地区：适合星空露营、晨跑、户外休息、低强度路线；遇到用户说“太累”“想随便走走”“不要听太多报告”，优先考虑这类区域和相邻活动。

## 餐饮与补给

地图左侧列出大量餐饮点，且会场北侧、东侧也有餐饮标注。推荐路线时：

- 12:00-13:30 和 17:30-19:30 不要安排过满。
- 如果用户在 A/B/C/D 主会场内连续活动，午餐建议在同区或相邻区解决。
- 如果用户要去星空露营、晨跑、户外活动，提醒提前带水、充电宝和外套。
- 如果用户说“第一次来”或“不熟场地”，安排路线时优先同楼层/同区，不要只按主题最优。

## 路线成本规则

- 同一房间/同一厅：可连续安排，但仍留 5-10 分钟。
- 同一 A/B/C/D 主会场簇内：建议留 10-20 分钟。
- 主会场到户外/东侧绿地/云栖之眼：建议留 20-30 分钟。
- 跨到餐饮区并返回：建议留 45-60 分钟，尤其饭点。
- 晚间活动后不要再安排高强度深度讨论，除非用户明确愿意。

## 推荐时如何使用

1. 先按日期和兴趣筛出候选。
2. 再按地图成本重排：同等匹配度时，同区、相邻区优先。
3. 为首次到场者保留“补给/找路/休息”空档。
4. 对高密度路线给出替代轻量选项，例如热带雨林、青春舞台、星空露营、户外活动。
