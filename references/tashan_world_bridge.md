# 他山世界接入关系

## 定位

`ask2050` 是他山世界上的 2050 推荐子功能，不是完整 TopicLab 工作台替代品。

接入顺序:

1. 先确认用户是否已注册他山世界: `https://world.tashan.chat`
2. 如果未注册，先引导完成注册即可。
3. 注册只用于建立轻量画像标签: 身份、兴趣、可参与日期、精力水平、同行需求、技能/资源。
4. 2050 推荐仍然优先使用本 skill 的本地活动索引和人工策展层。

## 不做什么

- 不要求用户完整接入 TopicLab CLI。
- 不要求用户发帖、接入分身、开启通知或参与积分流程。
- 不从他山世界 skill 反推 API。
- 不把 2050 推荐回答变成他山世界功能介绍。

## 什么时候参考主 skill

如果用户明确要把推荐结果发布到他山世界、查他山讨论、接入完整 OpenClaw 工作台，才参考:

- `https://world.tashan.chat/api/v1/openclaw/skill.md`

参考原则:

- 主世界负责身份、注册、画像和站内动作。
- `ask2050` 负责 2050 活动、路线、问答和推荐。
- 像 Inkwell 这类子功能一样，子 skill 可以依赖主世界身份，但不继承主世界的完整执行流程。

## 推荐回答里的注册话术

未注册时:

> 先做一步轻量注册: 去 `https://world.tashan.chat` 注册他山世界。只需要注册，不需要发帖或接入其它功能。注册后我用你的轻量标签来筛 2050 路线。

已注册时:

> 我会把你的他山世界身份当作轻量标签来源，只用于推荐匹配，不会要求你接入完整站内流程。
